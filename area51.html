The "LoB" or "Line of Business" Data-Parameter Requirements" is in DRAFT at this time.

OVERVIEW

The general idea for the parameters is to capture the data from the user as they enter, navigate, and exit the websites.

MANAGING BUSINESS PARTNERS:

Amanda D Celestin

STRATEGY:

This is used in Eloqua for identifying a user's interest area when someone fills out a promo page this is used to tag it and we can score them higher versus someone who just filled out like blog or a form on like a downloaded asset.

REQUIREMENTS

  1. The value shall be the domain name by default.
  2. The value shall be "reservations" if the user completes a form on reservations.ryder.com page.
  3. The value shall be "used-trucks" if the user completes a form on ryder.com/used-trucks page.
  4. The value shall be "supply-chain-management" if the user completes a form on ryder.com/supply-chain-management page.
  5. The value shall be "dedicated-transportation" if the user completes a form on ryder.com/solutions/dedicated-transportation page.
  6. The value shall be "solutions" if the user completes a form on ryder.com/solutions page.
  7. The value shall be "technology-innovations" if the user completes a form on ryder.com/technology-innovations page.
  8. A formula shall exists to identify the Lob
  9. The value shall include the path of the URL formatted as follows:
    example: ryder.com/solutions/dedicated-transportation becomes "solutions/dedicated-transportation"
